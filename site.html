Site web

Voici le lien du site web que nous avons réalisé : Lien
Ce site web permet de consulter les dispositifs recensés, et de pouvoir gérer ces dispositifs.
Il y a donc deux types d’utilisateurs qui peuvent avoir accès au site :

  • les simples visiteurs qui consultent librement les dispositifs
  • les modérateurs/gestionnaires qui gèrent la base de données de dispositifs

Gestion du site web

Une fois que les gestionnaires ou modérateurs du site se sont connectés, ils ont accès à différents formulaires, afin de gérer les dispositifs :

Les pages de consultation des dispositifs

Trois d’entre nous ont codé les pages PHP nécessaires à la consultation des différents dispositifs recensés au premier semestre. Le travail a été réparti comme il suit :

  • Marwa pour la page des déficiences et de la première catégorie
  • Émilie pour la page des sous-catégories
  • Alexia pour la page des dispositifs

Ensuite, nous avons mis notre travail en commun, nous avons commenté notre code et nous avons codé un fil d’Ariane, afin que le futur utilisateur puisse se repérer facilement.

Par la suite, nous avons envoyé nos pages PHP à iUserLab et nos tutrices afin de savoir si notre code convenait et pour savoir ce qu’ils attendaient de nous ensuite.


Les formulaires de gestion des dispositifs

Dans un premier temps, nous avons déterminé ensemble les différents formulaires qui seraient utiles aux gestionnaires, ainsi que toutes les fonctionnaliés à ajouter à notre site initial pour pouvoir disposer d’une connexion sécurisée inhérente au statut de gestionnaire.

Puis, dans un second temps, nous nous sommes réparties les formulaires à coder tout en continuant d’implémenter la base de données pour la finir. En parallèle, nous avons réalisé le site de soutenance et le guide d’utilisaton.


Le design du site web

Une fois les premières pages PHP codées, Marie a établi un premier design pour notre site. Elle nous l’a ensuite proposé et nous avons discuté ensemble des éléments qu’on voulait conserver et des détails à modifier.

Par la suite, nous avons conservé ce même design pour les pages des gestionnaires.
